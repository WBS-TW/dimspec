Welcome to the NIST database for identification of perfluoroalkyl substances (PFAS) by non-targeted analysis (NTA) using high resolution accurate mass spectrometry (HRAMS).

The goal of this project is to improve access to and utility of analytical data for the confidence dicovery, identification, and source-attribution of PFAS in environmental matrics.


The quality control check tool for the PFAS Mass Spectral database is for the evaluation of mass spectral data for general quality and for the generation of import files that can be directly incorporated into the database.

In order to use this application, the following must be done:

  • Raw data files must be converted to mzML (link here for directions)
  • Sample JSON files for each raw data file must be generated, using the Method Reporting Tool
