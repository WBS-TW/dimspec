Welcome to the NIST database for identification of perfluoroalkyl substances (PFAS) by non-targeted analysis (NTA) using high resolution mass spectrometry (HRMS).

The goal of this project is to improve access to and utility of analytical data for the confident discovery, identification, and source-attribution of PFAS in environmental matrics. It is powered by the data infrastructure for mass spectrometry (DIMSpec) project and includes curated HRMS spectral data (MS1 and MS2) for a collection of PFAS.

As part of that, the Mass Spectral Match (MSMatch) application is for matching user data (in mzML format) against this mass spectral database for matching analytes and fragmentation patterns. Full usage instructions can be found in the DIMSpec User Guide, or in the quick guide for web applications.
